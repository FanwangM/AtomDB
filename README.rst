AtomDB
======
|Python| |Github|

About
-----

AtomDB is a database of atomic and ionic properties.

Installation
------------

.. code-block:: shell

    python -m pip install -e .

Usage
-----

``python -m atomdb [-h] {generate,compile,query} args...``
